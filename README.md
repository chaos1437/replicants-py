Life simulation.

**Features:**
- *Save/load world state (--save_file)*
- *Minimal working GUI*
- *Interaction system(easy to add new types)*
- *Customizable world size*



**REQUIREMENTS:**
- *pygame*

*python3 -m pip install -r requirements.txt*



**For get help for additional parameters:**
    
*python3 main.py --help*
